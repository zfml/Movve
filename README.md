# Movve
## About
#### Fetch Movies from [TMDB MOVIE API](https://developer.themoviedb.org/reference/intro/getting-started) and can search ,view detail movie
## Built With
- Hilt Dependency Injection
- Retrofit
- Paging 3
- Jetpack Compose
